# CRUD de Clientes

Aplicação web que consome a API CrudCrud.

## Funcionalidades
- Cadastrar clientes
- Listar clientes
- Excluir clientes

## Como usar

1. Acesse https://crudcrud.com
2. Gere um endpoint
3. Substitua no arquivo scripts.js:

const API_URL = "https://crudcrud.com/api/SEU_ENDPOINT_AQUI/clientes"

4. Abra o index.html
